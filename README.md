# Maestría en Big Data & Business Intelligence – Repositorio Académico  
# Master in Big Data & Business Intelligence – Academic Repository

Este repositorio reúne el material de trabajo desarrollado a lo largo de la Maestría en Big Data & Business Intelligence de NEXT Educación.  
El propósito de este repositorio es **documentar, organizar y presentar los ejercicios, análisis y proyectos realizados en cada asignatura del programa**.

This repository contains the academic work developed during the Master in Big Data & Business Intelligence at NEXT Educación.  
Its purpose is to **document, organize, and present the exercises, analyses, and projects completed for each course of the program**.

---

## 1. Estructura general del repositorio  
## 1. General Repository Structure

El repositorio se organiza por asignaturas, manteniendo una carpeta independiente para cada curso o trabajo relevante.  
Cada carpeta incluye su propio README y, cuando corresponde, una licencia específica.

The repository is organized by course, maintaining an independent folder for each subject or major assignment.  
Each folder contains its own README and, when applicable, a specific license.

maestria-bigdata/
│
├── Asignatura_4/
│ └── ACT0504_HeartDisease/ # Proyecto de análisis estadístico
│
├── LICENSE # Licencia general del repositorio
└── README.md # Este documento

Cada proyecto dentro de una asignatura tiene su propia documentación interna, dependencias y licencia cuando corresponda.

Each project inside a subject folder includes its own internal documentation, dependencies, and license when required.
